#### Data Visualization
- Very important in Data Analysis
  - Exlore data
  - Report insights

```sql
import matplotlib.pyplot as plt

year = [1950, 1970, 1990, 2010]
pop = [2.519, 3.692, 5.263, 6.972]

plt.plot(year, pop)
plt.show()
```
